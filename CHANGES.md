# Изменения в этой версии PlayFound

## Реализовано в коде
- Обновлена mock-авторизация на `localStorage`.
- Регистрация создаёт обычный аккаунт игрока.
- Сессия сохраняется после reload до кнопки “Выйти”.
- Админ входит через `PlayFoundadmin` / `14886769`.
- Разработчиком можно стать только из аккаунта игрока на странице `/developers`.
- При создании developer-профиля имя становится `Имя Разраб`.
- Добавлена страница `/developer/games` для заявок разработчика.
- `/submit` доступна только developer-сессии.
- Добавлена страница `/profile`.
- В профиле можно менять аватарку, ник, цвет ника и описание.
- Добавлен индивидуальный ID игрока `PF-XXXXXXXX`.
- Добавление друзей работает по индивидуальному ID.
- Заявки в друзья принимаются или отклоняются в профиле под аватаркой.
- Добавлена mock-модерация мата в нике/описании.
- Добавлена mock-проверка ссылки на аватарку.
- Добавлены страницы `/clips`, `/forum`, `/jobs`, `/notifications`.
- Переработана админка: sidebar, операционный центр, роли, безопасность, валюты/языки, модерация.
- В каталог добавлены игры, похожие по настроению на Terraria/Stardew Valley: `Cavernkind`, `Greenvale Seasons`, `Starroot Hamlet`.
- `GameMedia` теперь умеет показывать реальные изображения из `public/games/...`, а не только градиенты.
- Добавлена подробная концепция `PLAYFOUND_PLATFORM_CONCEPT.md`.
- Добавлена инструкция `ADDING_GAMES.md`.

## Проверка
- `npm run typecheck` — успешно.
- `npm run build` в песочнице компилирует проект и генерирует страницы, но среда зависает на стадии `Collecting build traces`. Это похоже на ограничение контейнера/Next build tracing, а не на TypeScript-ошибку. Для Render оставлен обычный `npm run build`.
