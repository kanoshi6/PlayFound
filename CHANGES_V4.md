# PlayFound V4 fixes

- Убрано дублирование названия и цены внутри карточки игры: теперь картинка показывает только статус, название выводится один раз под баннером.
- Обложки и баннеры игр заменены на локальные SVG-постеры в `public/games/<slug>/`, чтобы карточки не выглядели как простые CSS-градиенты.
- Каталог переделан в сторону современной игровой витрины: крупные highlight-баннеры, промо-rail, фильтры и карточки с расширением/подъёмом при наведении.
- В шапке оставлены только игроко-ориентированные разделы: Главная, Каталог, Ролики, Форум. Вакансии убраны из основной навигации.
- Раздел `/jobs` теперь доступен только developer/admin-сессии. Обычный игрок получает экран доступа и переход на `/developers`.
- В footer вакансии показываются только разработчикам.
- Убраны UI-упоминания подписочной модели.
- Добавлена более явная анимация переходов между страницами через `PageTransition` и CSS `page-transition`.
- Раздел роликов получил лайки, сохранения и комментарии через `localStorage`.
- В профиле добавлен блок, где пользователь видит лайкнутые/сохранённые/прокомментированные ролики.
