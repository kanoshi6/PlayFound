# PlayFound Platform V3 — UX Fix Pass

## Исправлено по замечаниям

- Навигация стала короче и больше не ломается в две строки: в шапке оставлены только главные игровые разделы.
- "Стать разработчиком" убрано из шапки и перенесено в футер/страницу разработчиков.
- PlayFound сильнее позиционируется как платформа для игроков, а не как сайт только для разработчиков.
- Убран блок правил форума с главного экрана форума; вместо него добавлены тренды и горячие темы.
- Цвет ника теперь выбирается кругами без вывода hex-цифр.
- Аватарка загружается файлом с компьютера, а не URL-ссылкой; используется dataURL в localStorage.
- Заявки в друзья находятся в профиле рядом с аватаром, без лишней подписи "под аватаркой".
- Раздел роликов открывает вертикальный viewer при клике, а не просто показывает предпросмотр.
- Каталог переработан: spotlight-баннер, больше фильтров, цена в валюте, быстрый предпросмотр при hover.
- Убрана подписочная логика: на сайте оставлены бесплатные игры, разовые покупки, донаты и промокоды.
- Добавлены валюты: USD, EUR, RUB, GBP, PLN, UAH, CNY, JPY, KRW.
- Добавлены языки локализации игр: RU, EN, ES, DE, FR, PL, UK, ZH, JA, KO.
- Добавлены дизайн-темы: Dark Green, Pure Dark, Minimal Light, Street Style, Pixel 90s, Dark Neon, Fantasy Scroll, Arcade Club, Horror VHS, Soft Indie.
- Вернули анимации: reveal-card, interactive-card, pulse-glow, hover-preview, floaty.

## Важно

Это всё ещё client-only прототип. Данные профиля, друзей, заявок и настроек хранятся в localStorage. Для продакшена нужны backend, база данных, хранилище файлов и настоящая модерация.
